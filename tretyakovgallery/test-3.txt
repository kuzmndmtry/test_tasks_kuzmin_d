Заголовок 

Проверка корректной маски ввода e-mail в поле подписки на рассылку


Предусловия:
1. Перейти на https://www.tretyakovgallery.ru/
2. Перейти в футер главной страницы сайта

Шаги
1. Ввести в поле подписки на новости name@mail.ru
2. Дать согласие на обработку персональных данных
3. Кликом по кнопке --> в поле ввода e-mail отправить данные

ОР
1. Отображается сообщение об успехе "Вы были успешно подписаны на рассылку!"
2. Запрос ушел ручкой POST api/subs/add/?lang=ru с телом
{
	"email": "name@mail.ru"
}
3. Статус ответа 200 ОК. Тело:
{
	"data": {
		"success": true
	},
	"status": "",
	"errors": []
}
